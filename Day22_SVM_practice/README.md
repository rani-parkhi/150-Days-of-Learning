# Day 22 - Support Vector Machine (SVM)

## Objective

Learn and implement the Support Vector Machine (SVM) classification algorithm using the Breast Cancer and Wine datasets from Scikit-learn.

## Datasets Used

* Breast Cancer Dataset
* Wine Dataset

## Tasks Performed

* Loaded both datasets using Scikit-learn.
* Explored the datasets using Pandas.
* Split the data into training and testing sets.
* Applied feature scaling using `StandardScaler`.
* Trained two SVM models:

  * Linear SVM
  * RBF SVM
* Evaluated the models using:

  * Accuracy Score
  * Confusion Matrix
  * Classification Report
* Compared the performance of KNN, Logistic Regression, and SVM.

## Files

* `breast_cancer_svm.py` – SVM implementation on the Breast Cancer dataset.
* `wine_svm.py` – SVM implementation on the Wine dataset.

## Libraries Used

* pandas
* scikit-learn

## Concepts Learned

* Support Vector Machine (SVM)
* Linear Kernel
* RBF Kernel
* Feature Scaling
* Model Evaluation
* Model Comparison

## Result

Successfully implemented Linear and RBF SVM on the Breast Cancer and Wine datasets, evaluated their performance, and compared SVM with KNN and Logistic Regression.
