# Day 21 - K-Nearest Neighbors (KNN)

## Objective

Learn and implement the K-Nearest Neighbors (KNN) classification algorithm using two built-in Scikit-learn datasets: Iris and Wine.

## Datasets Used

* Iris Dataset
* Wine Dataset

## Tasks Performed

* Loaded both datasets using Scikit-learn.
* Explored the datasets using Pandas.
* Split the data into training and testing sets.
* Applied feature scaling using `StandardScaler`.
* Trained KNN models with different values of K (1, 3, 5, 7, and 9).
* Evaluated the models using:

  * Accuracy Score
* Predicted the class of new samples.
* Compared the performance for different values of K.

## Files

* `knn_iris.py` – KNN implementation on the Iris dataset.
* `knn_challenge.py` – KNN implementation on the Wine dataset.

## Libraries Used

* pandas
* scikit-learn

## Concepts Learned

* Classification
* Data Exploration
* Train-Test Split
* Feature Scaling
* K-Nearest Neighbors (KNN)
* Model Evaluation

## Result

Successfully implemented the KNN algorithm on both the Iris and Wine datasets and compared model performance across different values of K.
